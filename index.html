FileDragBox.js

Que es FileDragBox.js?

FileDragBox.js es una librería simple y minimalista que permite convertir un elemento HTML en un contenedor drag and drop para recibir archivos, opcionalmente puede adjuntar los archivos recibidos a un <input:file> para trabajar con formularios tradicionales.

Como Funciona?

Siguiendo la filosofía minimalista, FileDragBox.js no interfiere con la interfaz, usted tiene toda la libertad para darle el diseño a su elemento/caja donde se va a arrastrar y soltar, esta librería solo se encarga de darle la funcionalidad, dándole al elemento virtudes similares a un elemento <input:file> pero solo arrastrando los archivos a la caja.

Como Instalarla?

Forma #1. Descargando el archivo de la libreria 'FileDragBox.js' en local, o usando el CDN y agregandolo a su html.

Forma #2. Usando NPM.

Como usarla?

Paso #1: Usarlo es muy facil, si usted la descargo de forma local o la esta usando con CDN, agregue el <script:src> con la libreria a su html.

Si esta usando NPM, importe la libreria en su JavaScript.


Paso #2: Agregue un elemento en su html (al cual se le aplicara el drag and drop de archivos), a este elemento agréguele un 'id' o una 'clase' para identificar el elemento.


Paso #3: Por ultimo, cree una nueva instancia de la clase 'FileDragBox': new FileDragBox('selector', options). El constructor recibe dos argumentos, el selector del elemento al que se quiere aplicar el drag and drop (en este ejemplo el selector es el id '#dragbox'), y como segundo argumento las opciones.


Nota: La opción 'success:' recibe una función anónima la cual proporciona dos argumentos adentro de la función, el primero es el 'evento', y el segundo es el 'FileList' (colección de archivos) de los archivos que han sido arrastrados a la caja, en el caso del ejemplo, los que han sido arrastrados al elemento '#dragbox' (en el caso del ejemplo, cada que se arrastran archivos al elemento, se muestra en consola los archivos "contenidos" en la caja, osea los archivos que han sido almacenados en el FileList).


La opción 'accept' (opcional), recibe un array con los tipos de archivo permitidos en el drag and drop, si arrastra un archivo de un tipo que no este en el array, este no se agregara al FileList.


Usar sin input:file

Si usted no necesita usar un <input> de tipo 'file' asociado al elemento drag and drop, puede solo usar un elemento para arrastrar los archivos, y a través de la opción success usted recibe los archivos arrastrados, y puede hacer con ellos lo que necesite, por ejemplo, enviarlos a el backend o mostrarlos en el DOM, etc.


Importante, success se ejecutara cada que archivos son arrastrados y soltados en el elemento (en este caso el elemento '#dragbox').


Usar asociado a un input:file

Esta opción es perfecta para trabajar con formularios tradicionales. Para asociar un <input:file> dentro del drag and drop, debe de agregar la opción 'inputFile:', la cual recibe un string con el selector (class o id) de el <input:file> que desea asociar al elemento drag and drop.


Ahora cuando usted arrastre archivos dentro del elemento, estos de agregaran al 'FileList' del <input:file> asociado.


Si usted enviá el formulario de la forma tradicional, los archivos que arrastro al elemento drag and drop también se enviaran con el <input:file> asociado.


Cada que se arrastren archivos a la caja, también se despacha un evento del tipo 'change' en el input asociado, por lo que en este caso no es estrictamente necesario usar 'success:', podria usar un myInput.addEventListener('change', (e) => {}) en lugar de 'success:'.


Nota: La función 'success:' se disparara también cuando usted agregue archivos a través del <input:file> asociado.

Como borrar la lista de archivos?

Para borrar (resetear) la lista de archivos en el elemento drag and drop y en el input asociado puede usar el metodo clear().

Opciones

Opción Descripción Por Defecto
inputFile Recibe un string con el selector del input (de tipo file) a asociar con el elemento drag and drop. Null
(Sin Input Asociado)
accept Recibe un array con los tipos de archivo aceptados ['tipo/formato']:
  • ['image/jpeg', 'image/png'] (solo acepta imagenes jpeg, jpg, png)
  • ['image/*'] (solo acepta imagenes, en cualquier formato)
  • ['video/*'] (solo acepta videos)
  • etc.
 Null
(Cualquier Archivo)
success Recibe un callback, al cual se le proporcionan dos argumentos, el event (evento) y el FileList (lista con los archivos agregados).

Ejemplo: success: (event, FileList) => console.log(FileList);

Esta se ejecuta cada que se arrastran archivos al elementos drag and drop o se agregan elementos a través del input asociado.

Nota: si arrastro los arhivos recibira un evento de tipo 'drop', si lo hace a traves del input asociado, el evento sera de tipo 'input'. 		Null

Metodos

Método Descripción
clear() Borra (resetea) la lista de archivos agregados al FileList (borra los items del DataTrasfer y el input.value).

Lo puede usar cuando necesite borrar/limpiar los archivos agregados, después de enviar el formulario con JavaScript.

Note: si enviá el formulario de la forma tradicional no sera necesario, puesto que al enviarlo se recarga la pagina y la información del formulario se borra.
